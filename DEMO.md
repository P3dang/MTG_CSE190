Use this URL to access the website: https://mtg-cse190.onrender.com/

Once opened, it'll prompt you to sign in. Make an account and once logged in you'll be able to use this service. This app scan magic the gathering cards. If you're unfamiliar, you can search on google "magic the gathering cards" and there should be images of cards for you to try out. There is a video guide inside the Readme/

Project:
This is a magic the gathering scanner and deck builder app that allows the users to import photos of their own collection and have those cards be save in one location. There will be a deck builder agent added in the following submission.
